# 📝 TODO List App

This is a simple TODO List application built while learning React.  
The main goal of this project is to practice React fundamentals and improve my frontend development skills.

As they say, the more you practice, the better you get — and this project is part of that journey 🚀

---

## 🚀 Tech Stack

- ⚛️ React (with Vite)
- 🎨 Tailwind CSS
- 🟨 JavaScript (Frontend only)
- ❌ No backend used

---

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Track completion time
- Responsive UI
- Clean and simple design

---

## 🛠️ What I Learned

- React components and props
- useState hook
- Event handling
- Conditional rendering
- Project structure using Vite
- Styling with Tailwind CSS
- Creating a production build
